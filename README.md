# 🎓 Amasiko Group Registration Portal

A comprehensive web-based student registration and management system for Amasiko Training.

## 🚀 Quick Start

1. Open `index.html` in any modern browser
2. **Admin Login:** `Admissions@amasiko.co.za` / `Admissions@2026`
3. **Student:** Register from landing page

## ✨ Features

- **Admin:** Student management, course creation, financial blocking/clearing, dashboard stats
- **Students:** Self-registration, course enrollment, module selection, PDF proof generation
- **PDF Generation:** Automatic registration documents with jsPDF

## 🛠️ Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- jsPDF & jsPDF-AutoTable for PDF generation
- LocalStorage for data persistence
